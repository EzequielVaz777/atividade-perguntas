Trabalho de Web Design, HTML, CSS e JavaScript

  • Pergunta 1

1. O que é Web Design e qual sua relação com Front-End?

Web Design é a área responsável pela criação da interface visual de sites, cuidando do layout, das cores, tipografia e experiência do usuário. Ele se relaciona com o Front-End porque o front-end transforma o design em código usando HTML, CSS e JavaScript.

2. Cite três princípios de design que melhoram a experiência do usuário.

• Consistência
• Hierarquia visual
• Espaçamento adequado

3. Qual a diferença entre Wireframe e Protótipo?

Wireframe é um esboço simples da estrutura do site, focado no layout. Protótipo é uma versão mais avançada, interativa ou visual, que simula como o site vai funcionar.

4. Liste três ferramentas usadas para criar protótipos.

• Figma
• Adobe XD
• Sketch

5. Explique a função da tag <header> e dê um exemplo de uso.

A tag <header> representa o cabeçalho de uma página ou seção, normalmente contendo o título, menus ou logotipo. Exemplo: <header><h1>Meu Site</h1></header>.

6. Qual a diferença entre uma tag em bloco e uma tag inline?

Tags em bloco ocupam toda a largura disponível e começam em uma nova linha (ex.: <div>, <p>). Tags inline ocupam apenas o espaço do conteúdo e não quebram linha (ex.: <span>, <a>).

7. Como aplicar CSS externo em um documento HTML?

Basta criar um arquivo .css e ligar ao HTML com a tag: <link rel="stylesheet" href="style.css">.

8. Cite três propriedades CSS comuns e sua função.

• color: define a cor do texto.
• background-color: define a cor de fundo.
• margin: controla o espaçamento externo.

9. O que é DOM e como o JavaScript interage com ele?

DOM (Document Object Model) é a representação do HTML como uma estrutura de objetos. O JavaScript pode acessar e alterar esses objetos para modificar conteúdo, estilos e eventos da página.
